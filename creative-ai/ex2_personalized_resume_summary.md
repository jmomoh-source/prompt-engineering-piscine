## Role 1: DevOps Engineer

**Prompt:**
Create a 3-sentence professional summary for a DevOps Engineer skilled in Python and cloud systems.

**Output:**
DevOps Engineer specializing in Python for automation scripting, CI/CD pipeline development, and infrastructure as code using tools like Terraform and Ansible. Expert in cloud platforms including AWS, Azure, and Google Cloud, optimizing scalable, secure environments that support high-availability applications. Delivered 60% faster deployments and 99.9% uptime through proactive monitoring, container orchestration with Kubernetes, and cost-efficient resource management.



## Role 2: Teacher

**Prompt:**
Create a 3-sentence professional summary for a Teacher skilled in Physics and Chemistry with strong conversational skills and proficiency in English.

**Output:**
Passionate Physics and Chemistry Teacher skilled in delivering engaging, concept-driven lessons that demystify scientific principles through experiments, simulations, and real-world applications. Excel with strong conversational skills and fluent English proficiency, facilitating dynamic classroom dialogues, student mentoring, and clear communication with stakeholders to build confidence and collaboration. Proven ability to boost academic performance and ignite lifelong curiosity in STEM among diverse learners.




## Comparison and Analysis

The DevOps summary uses technical jargon like CI/CD, Kubernetes, and Terraform, with quantifiable metrics “60% faster deployments, 99.9% uptime” to demonstrate value. 

While the Teacher summary emphasizes more on soft skills and emotional impact using words like "passionate," "engaging," and "ignite curiosity" focused on student outcomes rather than numbers. 

This shows that the AI automatically adapts tone, vocabulary, and emphasis based on the role specified in the prompt, technical roles get metrics and tools, while education roles get impact focused, human centered language.

## 📖 Overview
This quest explores how well-designed prompt patterns can replace vague, open-ended requests with structured templates that produce reliable and targeted results.  
Key themes:
- Reusable prompt structures for different tasks  
- Instructional, comparative, QA, and data extraction patterns  
- Transforming weak prompts into structured, repeatable templates
